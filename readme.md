# レーザー除草ロボットシステム

## プロジェクト概要

本プロジェクトは、**除草剤の過剰使用による環境汚染**と**農業における深刻な労働力不足**という緊急課題を解決するため、**精密農業（Precision Agriculture）**を目指した**自律走行除草ロボット**を開発したものです。

システムの主な構成：
- **人工知能（AI）** + **コンピュータビジョン** をエッジデバイス（Jetson Nano）で実装
- 深層学習モデル **YOLOv8-segmentation** によるリアルタイム雑草検出
- **ステレオカメラ** を用いた深度計算と正確な3D座標取得
- 化学除草剤を一切使用せず、**高出力レーザー** で雑草を除去
- **Webサーバー（Java Servlet + Apache Tomcat）** による遠隔監視・制御

実証実験の結果：ロボットは安定した走行を行い、雑草の検出・処理が効率的で、Webインターフェースは直感的かつ視覚的に優れており、作業履歴の追跡とパフォーマンス統計を提供します。

---

##  主な目的

- 畝に沿って安定走行する自律ロボットを構築し、Jetson Nano上でリアルタイム雑草検出を実現
- **ステレオビジョン** を活用し、2D画像座標 → 3D空間座標（x, y, z）への変換を行い、レーザーの正確な照射を可能に
- **Web監視システム**（Java Servlet + JSP + Tomcat）を構築し、ハイブリッド通信（**TCPソケット**：低遅延制御 / **HTTP RESTful API**：大容量データ転送）を実装
- MySQLデータベースで作業セッション、除草履歴、統計情報を保存

## 技術構成・システムアーキテクチャ

### 1. ハードウェア（Hardware）
- **中央処理装置**：NVIDIA Jetson Nano（AI性能472 GFLOPS、4GB RAM、128コアCUDA GPU）
- **マイコン**：Arduino（Pan-Tiltサーボ、DCモーター、レーザーリレー制御）
- **センサー**：
  - 2台のウェブカメラ XWF-1080P（ステレオカメラシステム構築）
  - TCRT5000ラインセンサー（ライン追従・距離計測）
  - デジタルコンパス + エンコーダー
- **アクチュエーター**：
  - DCモーター + 車輪（直径34cm）
  - Pan-Tiltサーボ + 高出力レーザー
- **内部通信**：UART（Jetson ↔ Arduino）、USB 3.0/CSI（カメラ）

**ハードウェアブロック図**（報告書14-17ページ参照）

### 2. ソフトウェア・AI
- **雑草検出**：YOLOv8-segmentation（ミントと雑草700枚のデータセットで学習）
  - 学習：75%（525枚）、検証：13%（90枚）、テスト：12%（85枚）
  - バッチサイズ = 8
  - 結果：mAP@0.5（Box）≈85%、Mask ≈80%、F1スコア最高値はconfidence 0.514～0.515
- **ステレオビジョン**：
  - OpenCV `stereoCalibrate` によるチェッカーボードキャリブレーション
  - Semi-Global Matching（SGM）でDisparity Map生成
  - 深度計算式：\( z = \frac{b \cdot f_x}{d} \)（b = ベースライン、d = 視差）
- **座標処理**：セグメント重心をK-meansで算出 → Inverse Kinematicsでサーボ角度計算 → レーザー照射
- **ハイブリッド通信**：
  - **TCPソケット**：リアルタイム制御（Start/Stop/Pause、心拍、状態更新）
  - **HTTP RESTful API**：証拠画像・統計情報のアップロード

### 3. Webサーバー・管理システム
- Apache Tomcat + Java Servlet + JSP
- MySQLデータベース（ER図：報告書36ページ）
- 主な機能：
  - ロボット一覧・リアルタイム接続状態表示
  - セッションごとの除草履歴（画像＋座標）
  - 統計ダッシュボード（除草数、カバー率、レーザー照射点数）
  - 将来的なビデオストリーミング対応

##  実証実験結果（要約）

- **Jetson Nano性能**：YOLOv8-segmentationをリアルタイムで安定動作
- **レーザー精度**：非常に高い（詳細評価：報告書39ページ）
- **AIモデル**：Precision/Recall >80%、mAP@0.5良好（混同行列：報告書41ページ）
- **Webインターフェース**：使いやすく同期性が高い（画面例：報告書36-37ページ）
- **レーザー照射点数**：セグメント面積＋レーザー影響円＋K-meansで最適化（省電力かつ最大除草効果）
